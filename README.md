# Dummy-ODL
This project provides a simple example of using a dummy ODL file 
## Usage
1. Copy `dummy.odl` to your Wi-Fi access point.
2. Build the data model using `amxrt -D ./dummy.odl`.
3. Query the data model using `ubus-cli Dummy.?`
